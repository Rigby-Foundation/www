# The Rigby Foundation

![Rigby Foundation Banner](public/rigbysplash.png)

Building open, secure, and decentralized tools for a private internet. We believe in code that protects users, not exploits them.

## Overview

This is the official website for The Rigby Foundation, built with modern web technologies to showcase our mission and projects.

### Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: [GSAP](https://gsap.com/) (GreenSock Animation Platform)
- **Smooth Scroll**: [Lenis](https://lenis.studiofreight.com/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/) (FontAwesome 6)

## Getting Started

To run this project locally:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Rigby-Foundation/www.git
    cd www
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # or
    bun install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    # or
    bun dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
